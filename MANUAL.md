## 1. ブログを起動する

### 手順

1. **ターミナル**を開く
   - Launchpad（ロケットのアイコン）→「ターミナル」で検索
   - または Spotlight（`⌘ + スペース`）→「ターミナル」と入力
2. 以下のコマンドを入力して Enter を押す：

```
cd ~/MEBAE-blog && ./start-blog.sh
```

3. 自動的にブラウザが開き、**管理画面**が表示されます 🎉

> 💡 ブラウザが自動で開かない場合は、以下のURLを手動で開いてください：
> **http://localhost:1313/admin.html**

## 3. 新しい記事を書く

### 手順

1. 管理画面で **「✏️ 新しい記事を書く」** ボタンをクリック

2. 以下の項目を入力します：

| 項目 | 説明 | 入力例 |
|------|------|--------|
| 📝 記事タイトル | 記事のタイトル | コストコで買ったおすすめ商品！ |
| 📁 ファイル名 | 空でOK（自動生成されます） | |
| 📅 日付 | 今日の日付が自動で入ります | |
| 📂 カテゴリー | 記事のジャンルを選択 | 🛒 Costco |
| 🏷️ タグ | キーワードを入力してEnter | コストコ、買い物 |
| 📄 説明文 | Google検索で表示される説明 | コストコのおすすめ商品紹介 |
| ✍️ 本文 | 記事の内容を自由に書く | （下記参照） |

3. 本文を書く

### 本文の書き方（かんたんMarkdown）

```
## 見出し（大きい文字になります）

普通の文章はそのまま書けます。

**太字にしたい文字**

- リスト項目1
- リスト項目2
- リスト項目3

[リンクテキスト](https://example.com)
```

### 便利ボタン

本文の上にあるボタンを使うと、かんたんに入力できます：

| ボタン | 機能 |
|--------|------|
| 📷 写真 | 写真を挿入するコードを追加 |
| 🖼️ ギャラリー | 複数の写真をまとめて表示 |
| 🎥 YouTube | YouTube動画を埋め込み |
| 🛒 Amazon | Amazon商品リンクを追加 |
| 📌 見出し | 見出し（大きい文字）を追加 |
| **B** 太字 | 太字テキストを追加 |
| 🔗 リンク | リンクを追加 |

## 4. 写真をアップロードする

### 手順

1. 記事編集画面の **「📸 写真アップロード」** エリアを探す

2. 写真を追加する方法は2つ：
   - **方法A**: 写真ファイルをドラッグ＆ドロップ（つかんで放す）
   - **方法B**: エリアをクリックして、ファイルを選択

3. アップロードされた写真がサムネイル（小さい写真）で表示されます

4. **写真をクリック**すると、本文に自動で挿入されます 📷

> ⚠️ **注意**: 写真をアップロードする前に、**タイトル**を入力してください。
> タイトルから自動でフォルダ名が作られます。

> 💡 **ヒント**: 複数の写真を一度にドラッグ＆ドロップできます！

## 5. 記事をプレビューする

公開する前に、記事がどのように表示されるか確認できます。

### 手順

1. 記事を書いたら **「💾 保存する」** ボタンをクリック
2. **「👀 プレビュー」** ボタンをクリック
3. 新しいタブでブログの表示が確認できます

> 💡 保存しないとプレビューできません。まず保存してからプレビューしてください。

## 6. 記事を公開する

記事をインターネットに公開して、誰でも見られるようにします。

### 方法A: 記事を書きながら公開

1. 記事を書き終わったら **「🚀 保存して公開する」** ボタンをクリック
2. 「公開中...」と表示されるので少し待ちます
3. 「🎉 公開完了！」と表示されたら成功です！

### 方法B: 記事一覧画面から公開

1. 記事一覧画面の下にある **「🚀 サイトに公開する」** ボタンをクリック
2. 少し待つと公開されます

> 📝 公開後、実際のサイト（https://mebae-california.com）に反映されるまで
> **1〜2分**ほどかかります。

---

## 7. 記事を編集する

すでに書いた記事を修正したい場合：

1. 管理画面の **記事一覧** から、編集したい記事をクリック
2. 記事の内容が表示されるので、修正する
3. **「💾 保存する」** をクリック
4. 公開中の記事の場合は **「🚀 サイトに公開する」** で反映

---

## 8. 記事を削除する

1. 管理画面の記事一覧で、削除したい記事の **🗑️ ボタン** をクリック
2. 確認ダイアログが表示されるので **「OK」** をクリック
3. 削除後、**「🚀 サイトに公開する」** で公開サイトからも消えます

> ⚠️ 削除した記事は元に戻せません。慎重に行ってください。

---

## 9. ブログを終了する

ブログの管理が終わったら：

1. ターミナルの画面に戻る
2. **`Ctrl + C`**（Controlキーを押しながら C）を押す
3. 「👋 おつかれさまでした！」と表示されて終了します

> ブラウザのタブは閉じるだけでOKです。

## 10. よくある質問

### Q: ブラウザに「接続できません」と表示される

→ ブログが起動していません。ターミナルで以下を実行してください：

```
cd ~/MEBAE-blog && ./start-blog.sh
```

### Q: 「API未接続」と表示される

→ 管理サーバーが起動していない可能性があります。
   一度ターミナルで `Ctrl+C` を押してから、再度 `./start-blog.sh` を実行してください。

### Q: 写真がアップロードできない

→ 先に **記事タイトル** を入力してから写真をアップロードしてください。

### Q: 公開ボタンを押してもサイトに反映されない

→ 反映まで1〜2分かかります。少し待ってからブラウザを更新（⌘+R）してください。

### Q: 間違えて記事を公開してしまった

→ 記事を編集して **「📋 下書きとして保存」** にチェックを入れ、
   保存してから「🚀 サイトに公開する」を押すと非公開になります。

### Q: 記事の順番を変えたい

→ 記事は日付の新しい順に表示されます。日付を変更すると順番が変わります。

### Q: カテゴリーを追加したい

→ Ethanさんに相談してください。設定ファイルの変更が必要です。
