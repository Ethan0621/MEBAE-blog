# 🌸 めばえのカリフォルニアブログ - 運用ガイド

## 📝 記事を投稿する方法（3つ）

---

### 方法① ブラウザの管理画面で書く（おすすめ！）

1. ターミナルで `./start-blog.sh` を実行（管理サーバー＆プレビューが同時に起動）
2. ブラウザで管理画面が自動的に開きます
3. 「✏️ 新しい記事」でタイトル・カテゴリー・タグ・本文を入力
4. 写真はドラッグ＆ドロップでアップロード → サムネイルクリックで挿入
5. 「� 保存する」で記事を保存
6. 「🚀 サイトに公開する」でインターネットに公開！

---

### 方法② スクリプトで作る（ターミナル）

```bash
./new-post.sh
```

質問に答えていくだけで記事のテンプレートが作成されます。

---

### 方法③ VS Code で直接書く

1. `content/posts/` フォルダを右クリック → 「新しいファイル」
2. ファイル名は `記事名.md`（例: `costco-april-2026.md`）
3. 以下のテンプレートをコピーして書き始める：

```markdown
---
title: "記事タイトル"
date: 2026-04-11
draft: false
categories: ["アメリカ生活"]
tags: ["ロサンゼルス"]
description: "記事の説明"
image: ""
---

ここに記事を書く
```

## 📸 写真の使い方

### 管理画面から写真を入れる方法（おすすめ！）

1. 記事の編集画面を開く
2. 「📸 写真アップロード」エリアに写真をドラッグ＆ドロップ、またはクリックして選択
3. アップロード完了後、以下のどちらかの方法で本文に挿入：

| 方法 | やり方 |
|------|--------|
| **方法A** | アップロード後に表示される**サムネイルをクリック** |
| **方法B** | 本文エリア上の**「📷 写真を挿入」ボタン**をクリック → 写真選択画面から選ぶ |

> 💡 iPhoneの写真アプリから直接ドラッグ＆ドロップもOK！HEIC形式にも対応しています。

### 複数写真をギャラリーで表示

1. 写真を複数枚アップロードする
2. 本文エリア上の**「🖼️ ギャラリー」ボタン**をクリック
3. ギャラリーに入れたい写真を**複数選択**（クリックで✅マーク）
4. 「挿入する」ボタンを押す → ギャラリーが自動生成されます

### 写真の保存場所（参考）
- 写真は自動的に `static/images/記事のスラッグ/` フォルダに保存されます
- 手動で保存する場合は `static/images/` フォルダに保存してください

## 🎥 YouTube動画の入れ方

1. YouTubeに動画をアップロード
2. 動画のURLから **ID** をコピー
   - `https://youtu.be/ABC123` → IDは `ABC123`
   - `https://www.youtube.com/watch?v=ABC123` → IDは `ABC123`
3. 本文エリア上の **「🎥 YouTube」ボタン** をクリック
4. 動画IDとタイトルを入力 →「挿入する」を押す

---

## 🛒 Amazon商品リンクの入れ方

1. Amazonの商品ページを開く
2. URLの中の **ASIN**（10桁の英数字）をコピー
   - 例: `amazon.com/dp/B00FLYWNYQ/...` → ASINは `B00FLYWNYQ`
3. 本文エリア上の **「🛒 Amazon」ボタン** をクリック
4. 商品名・ASIN・おすすめポイントを入力 →「挿入する」を押す

---

## ✍️ 本文の編集ボタン

本文エリアの上にあるボタンで、さまざまな書式を簡単に挿入できます。

| ボタン | 機能 |
|--------|------|
| 📷 写真を挿入 | アップロード済み写真から選んで挿入 |
| 🖼️ ギャラリー | 複数写真を選んでギャラリーを作成 |
| 🎥 YouTube | 動画IDを入力して動画を埋め込み |
| 🛒 Amazon | 商品名・ASINを入力して商品リンクを挿入 |
| 📌 見出し | テキストを入力して大きな見出しを挿入 |
| **B** 太字 | テキストを選択してから押すと太字に変換 |
| 🔗 リンク | テキストとURLを入力してリンクを作成 |

> 💡 太字は、先にテキストを選択してからボタンを押すと、選択部分が太字になります。

## 🗑️ 記事を削除する

1. 「📋 記事一覧」を開く
2. 削除したい記事の右側にある **🗑️ ボタン** をクリック
3. 確認ダイアログで「OK」を押す
4. 「🚀 サイトに公開する」で本番サイトからも削除されます

> ⚠️ 削除した記事は元に戻せません。間違って削除しないよう注意してください。

## 📂 ホームのカテゴリーを管理する

トップページに表示されるカテゴリーボックスの追加・削除・並び替えができます。

### 管理画面から操作する方法（おすすめ！）

1. ターミナルで `./start-blog.sh` を実行
2. ブラウザで **http://localhost:1313/admin.html** を開く
3. ヘッダーの「📂 カテゴリー」ボタンをクリック
4. 以下の操作ができます：

| やりたいこと | 操作 |
|------------|------|
| 並び替え | ☰ マークをつかんでドラッグ＆ドロップ |
| 追加 | 「＋ カテゴリーを追加」ボタンをクリック |
| 削除 | 各行の右側の ✕ ボタンをクリック |
| 絵文字の変更 | 左側の絵文字欄をクリックして入力 |
| 名前・URLの変更 | 各欄を直接クリックして編集 |

5. 「💾 保存する」をクリック
6. 「🚀 サイトに公開する」で本番サイトに反映

### URLの形式

カテゴリーのURLは以下の形式で入力します：
```
/categories/カテゴリー名/
```
例：`/categories/アメリカ生活/`、`/categories/costco/`

> 💡 カテゴリーを追加した場合は、対応する `content/categories/カテゴリー名/_index.md` ファイルも必要です。
